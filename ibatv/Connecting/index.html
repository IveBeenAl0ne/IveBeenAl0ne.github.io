How to Connect.

For simplicities sake, we use a program called "Radmin VPN" to connect and play.
If you need a copy of it, you can get it at https://www.radmin-vpn.com.
The connection Information Is located in the Alone Dreamers Discord,
assuming that you have the valid role for it. If not let us know.
We do ask that you change your username in Radmin to the same as your Discord server name.

If we can the servers we host will be under the "Local" tab in the game servers tab.
